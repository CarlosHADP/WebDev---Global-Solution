Seção 1

Problema

Os oceanos enfrentam destruição de habitats, poluição e mudanças climáticas, impactando ecossistemas marinhos e a economia global.

Solução

Nossa solução consiste em uma boia com Arduino e sensores para monitorar CO₂, combatendo impactos negativos na vida marinha.

Seção 2

Tecnologias que Iremos Utilizar

Utilizaremos paginas Web para divulgar artigos de concientizacao e informacoes do nosso projeto para concientizar as pessoas, empresas que utilizam o mar.

Arduinos, para captar dados ambientais críticos, como temperatura da água, níveis de pH e poluição sobre o mar e nos retornar esses dados para tomarmos atitudes devidas.

Python para manipular os grandes bancos de dados fornecidos pelos nossos arduinos e polir esses dados para serem mostrados com mais efetividade nas nossas paginas Web.
